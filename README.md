# port-scanner
Light weight bash port scanner 

Usage:
```
chmod +x ./script.sh
./script.sh -i <ip> -p <High port cap>
```
Example:
```
root@:~/# ./script.sh -i 10.10.10.10. -p 10000                                                                                                                                                                                                             
Port 53 is open
Port 80 is open
Port 139 is open
Port 443 is open
Port 445 is open
root@:~/#
```
